# Повідомлення про вразливості

Щоб повідомити про проблему безпеки, будь ласка, скористайтеся сайтом
[https://g.co/vulnz](https://g.co/vulnz). Ми використовуємо g.co/vulnz для
прийому звернень, а координацію та розкриття інформації проводимо тут, на GitHub
(включаючи GitHub Security Advisory). Команда безпеки Google відповість протягом
5 робочих днів після вашого звіту на g.co/vulnz.

[GitHub Security Advisory]:
  https://github.com/google-gemini/gemini-cli/security/advisories
